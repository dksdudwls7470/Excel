# Excel
#### 학습정리
----------------
#### CHAPTER1 엑셀기초
```
1. 오피스 2016이란?
엑셀은 마이크로소프트 오피스 패키지의 하나로서 스프레드시티 프로그램이다. 엑셀을 사용 하여 예산 관리, 
판매 분석, 예측, 차트 작업등이 가능하다.
```
```
2. 엑셀에서 할 수 있는 일
엑셀에서 데이터의 행과 열을 입력한 후 다양한 방법으로 데이터를 다룰 수 있다. 엑셍을 사용하는 이유는 
  가. 데이터를 빠르고 정확하게 입력
  나. 데이터를 쉽게 재계산
  다. 광범위한 내장 함수 활용
  라. 정보의 외양을 변경
  마. 데이터베이스 관리
  바. 차트 생성
  사. 가상 분석을 수행
```
```
3. 엑셀 VS 엑세스 VS. 통계패키지
엑셀은 관계 데이터베이스가 아니다. 엑셀은 데이터를 정렬하고 검색할 수 있지만 복잡한 데이터 관리를 위해서는 엑세스와 같은 데이터베이스
소프트웨어를 사용해야 한다. 일반적으로 엑세서는 데이터베이스를 다루고 질의하는데 훨씬 더 정교한 도구들을 제공한다. 그러나 엑셀은 액세스보다
데이터 분석과 차트 작성에 매우 강력하다. 엑셀과 액세스 사이에 파일을 가져오거나 내보내는 것이 가능하다. 에는 액세스 파일을 엑셀로 가져와서
엑셀을 치트 기능 또는 계산 기능을 활용하고, 엑셀 파일을 액세스로 내보내서 액세스의 좀 더 강력한 데이터베이스 기능을 활용 할 수 있음.
```
```
4. 주요 용어
스프레드시트는 일반적인 용어이다. 엑셀에서 사용하는 용어는 통합 문서와 워크시트이다.
  가. 통합 문서 : 엑셍을 사용하여 만든 파일을 통합문서라고 부른다
  나. 워크시트 : 각 통합 문서는 하나 이상의 워크시트를 포함한다. 워크시트는 통합 문서 내의 독립적인 단위이다. 통합 문서와 워크시트 간의 관계
      는 공책을 구성하는 페이지에 비유할 수 있다.
```
```
5. 엑셀 2016의 새로운 기능
엑셀 2016에 새로 추가된 주요 기능
  가. 새로운 종류의 치트
  나. 피벗 테이블 슬라이서
  다. 쉽게 통합 문서를 pdf로 변환
  라. 동료들과 공유하고 협동하는 기능
  마. 도움말을 활용하는 새로운 방법
  바. 함수 마법사를 호출하는 새로운 방법
  사. 예측 시트
```
